tax_group_tax_band

Peril_Type records used to be linked to ONE tax_type record.

In order to be able to join a Peril_Type to SEVERAL tax_types, a new Tax_group table was created. Severl tax_types can be linked to a tax_group, using "tax_group_tax_band" as a 2 fields link table.

Peril_types are then linked to a tax_group.

Logical database: BackOffice
Component team: Pure For Underwriting
Software module: Policy
