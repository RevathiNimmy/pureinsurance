DOC_doc_keyword

Holds details of the keywords associated to a document. Links the DOC_document and DOC_keyword tables.

Logical database: DocuMaster
Component team: DocuMaster
Software module: DocuMaster

keyword_id integer null FK, I ð DOC_keyword.keyword_id Relates to the doc_keyword table.
user_name varchar(16) not null   Username of the user that added the keyword to the document.
create_date datetime not null   Date on which the keyword was added to the document.
