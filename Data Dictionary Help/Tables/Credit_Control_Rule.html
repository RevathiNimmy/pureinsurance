Credit_Control_Rule

Defines an individual rule that the Credit Control system will use for a particular Branch, Business Type (NB/REN/MTA), and/or Frequency (for Instalments). This will decide which Credit_Control_Steps will be used.

Logical database: Orion
Component team: Pure Accounts
Software module: Accounts
