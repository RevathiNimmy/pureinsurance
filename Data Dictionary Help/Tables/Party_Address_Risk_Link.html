Party_Address_Risk_Link

Specifies which Address associated with a particular Party applies to a particular Risk Group.

Logical database: BackOffice
Component team: Pure For Broking
Software module: Party

Definition

Name Data Type Defaults/Rules Constraints Description
party_cnt integer not null PK, FK, I ð Party.party_cnt party
address_cnt integer not null PK, FK, I ð Address.address_cnt address
risk_group_id integer not null PK, I This field is used to link addresses to risk types for mailshot purposes. This can be achieved via Insurer Maintenance {Address Tab - Risk Button}.
