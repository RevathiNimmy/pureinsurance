StructureTree

The master Tree used by the Account Explorer. This is a self-referencing table to represent nodes within the tree with their children and parents.

Logical database: Orion
Component team: Pure Accounts
Software module: Accounts

Definition

Name Data Type Defaults/Rules Constraints Description
node_id integer identity not null PK Unique ID - automatically generated by database
company_id integer not null I company id = source id
mapping_id integer null FK, I ð Mapping.mapping_id mapping
account_id integer null FK(L), I ð Account.account_id account
element_id integer null FK, I ð Element.element_id element
parent_node_id integer not null FK, I ð StructureTree.node_id parent node
core_node tinyint null I Core Node? (NULL = added by the user, 1 = added at system initialisation). For future expansion - this will allow Pure to stop the user from removing core accounts or folders.
