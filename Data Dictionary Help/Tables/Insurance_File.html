Insurance_File

Policy version history (the main policy record is stored in Insurance_Folder). A new Insurance_File row is created for a new quote, a "go live", an MTA, a cancellation, or a renewal (including "what if" scenarios). All other changes result in the "current" Insurance_File row being updated, and a new event generated (see Event_Log and Event_Insurance_File).

Logical database: BackOffice
Component team: Pure Core Back-Office
Software module: Policy
