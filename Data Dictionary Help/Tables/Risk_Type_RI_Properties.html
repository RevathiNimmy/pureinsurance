Risk_Type_RI_Properties

Which properties are important when determining ri limits.

NOTE: This table only has a relationship with the gis_property table via the gis_property_id column. This is incorrect and should really be the gis_object_id/gis_property_id compund key from the gis_property table. This will work as long as there are no duplicate gis_property_id values within the gis_property table. Product Builder will always assign an incrementing value when creating gis_properties but conflicts can occur if GeminiII is installed. GII uses hard coded object and property id values and therefore clashing of property_id values will occur at some point.
