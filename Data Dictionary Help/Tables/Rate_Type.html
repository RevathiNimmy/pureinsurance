Rate_Type

Defines the rate types valid within rating. Currently these are:
O - No Rating
V - Value
Q- Quantity X rate
P - Per 1000
C - Percentage
M - Per Million
T - Percentage Of Running Total

This table is not maintainable by users.

Logical database: BackOffice
Component team: Pure For Underwriting
