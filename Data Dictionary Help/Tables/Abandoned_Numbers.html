Abandoned_Numbers

When a quote, policy or claim has a number auto generated and is then abandoned, this number is stored in this table. But only if the re-use abandoned numbers flag is set on the numbering scheme table.

Logical database: BackOffice
Component team: Pure For Underwriting
Software modules: Claims, Policy

Definition

Name Data Type Defaults/Rules Constraints Description
numbering_scheme_id integer not null PK, I The id of the scheme that this abandoned number comes from.
abandoned_number varchar(20) not null PK abandoned number

Type Column(s) Referenced Table Referenced Column(s) Description

I numbering_scheme_id
