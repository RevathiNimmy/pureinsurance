GIS_Polaris_Mapping

GIS Polaris Mapping is used to hold the mappings between Gis properties and Polaris property names. It is not used at runtime, it is used to generate an xsl which translates the dataset into an xml suitable for the polaris XRTE to process and hence produce a quote output xml. At present this is only used by Its4Me but may be utilised by Towergate and GII some time in the future.

Logical database: GIS
Component team: Pure Architecture
Software module: GIS
