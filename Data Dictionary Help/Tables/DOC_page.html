DOC_page

Holds the individual page details and is linked to the DOC_document table.

Logical database: DocuMaster
Component team: DocuMaster
Software module: DocuMaster

Definition

Name Data Type Defaults/Rules Constraints Description
page_name varchar(254) not null AK Name for the page. This is the filename.
doc_num integer not null PK, FK, I ð DOC_document.doc_num Foreign key relating to the doc_document table.
page_num integer not null PK Page number.
page_type varchar(254) not null   The type of page.
DOC = Document
TIF = TIF image
create_date datetime not null   Date the page was created (added to the DME system).
page_size integer not null   Size (in bytes) of the document.
volume_id integer not null FK ð DOC_volume.volume_id Relates to the doc_volume table to indicate which volume the document is stored on.
