DOC_doc_user

Holds information and permissions for the DocuMaster users. Links to the PMUserr table.

Logical database: DocuMaster
Component team: DocuMaster
Software module: DocuMaster

Definition

Name Data Type Defaults/Rules Constraints Description
user_id smallint not null PK Primary key for the table.
access_level tinyint not null   The users access level.
user_name varchar(255) null   user name
home_folder_num integer null   Relates to doc_folder. This is the folder that is the users home folder. This is the first folder that gets shown when they start DME.
retired char(1) not null   Similar to is_deleted. Bit flag to indicate if the user is still active on the system.
