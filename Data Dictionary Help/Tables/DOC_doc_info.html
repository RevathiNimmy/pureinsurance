DOC_doc_info

Holds extra info about the document.

Logical database: DocuMaster
Component team: DocuMaster
Software module: DocuMaster

Definition

Name Data Type Defaults/Rules Constraints Description
doc_num integer not null PK, FK, I ð DOC_document.doc_num Primary key for the table. Automatically generated by the database.
expiry_date datetime null   The expiry date for the document.
scan_user varchar(16) null   The username for the user that scanned the document (if it was imported via scanning).
doc_date datetime null   The date for the document. Not necessarily the create date, as it can be altered by the user when scanning.
last_user varchar(16) not null   The last user to access the document.
last_date datetime not null   The date the document was accessed last.
