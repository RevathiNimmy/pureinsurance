DOC_keyword

Holds the current keywords available.

Logical database: DocuMaster
Component team: DocuMaster
Software module: DocuMaster

Definition

Name Data Type Defaults/Rules Constraints Description
keyword_id integer identity not null PK Unique ID - automatically generated by database
keyword varchar(30) not null   The actual keyword (just a single word...)
deleted char(1) null   Bit flag to indicate if the keyword has been "deleted". i.e. is no longer available for use.
