DOC_device

Holds the location of the document store.

Logical database: DocuMaster
Component team: DocuMaster
Software module: DocuMaster

Definition

Name Data Type Defaults/Rules Constraints Description
device_id integer not null PK Primary key for the table.
device_name varchar(30) null   Name for the device. Not displayed anywhere, just for reference.
server_unc varchar(50) not null   This can either be a path to a UNC share on a server, or a local path.
share_name varchar(20) not null   The share_name gets appended to the server_unc to form the data folder path.
The directory from the doc_volume table then gets appended to form the entire path to the DME data storage folder.
drive varchar(20) null   No longer used. Can be left either blank or null.
