PMLogicalDatabase

List of installed logical databases. Used by the Database Installer.

Logical database: Architecture
Component team: Database
Software module: Database Installer
System table

Definition

Name Data Type Defaults/Rules Constraints Description
pmlogicaldatabase_id integer not null PK Unique ID
name varchar(30) not null AK Internal name of logical database, must be unique. Originally defined in the config.xml file on the install CD, and also used as the name of the script folder for this database.
version varchar(30) not null   Version - the installer expects plain digits separated by full stops, e.g. "1.8.6.0234"
install_date datetime not null   Date last created or upgraded
