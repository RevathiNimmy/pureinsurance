GIS_Policy_Link

Links dynamic data model tables to the rest of the system. Each row in this table has a one-to-one link with a row in the top-level table (<data model>_Policy_Binder) for that particular data model. Also, the system assumes that the IDs of GIS_Policy_Link and <data model>_Policy_Binder are identical. For example:
GIS_Data_Model(gis_data_model_id, code, description) = (6, 'TEST', 'Test Data Model')
GIS_Policy_Link(gis_policy_link_id, gis_data_model_id) = (1027, 6)
TEST_Policy_binder(TEST_Policy_binder_id, gis_policy_link_id) = (1027, 1027)

Logical database: GIS
Component team: Pure Architecture
Software module: GIS
